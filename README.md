# 🎮 Games Haven  
A modern Django-based gaming platform built during the Tuwaiq Academy Python Web Development Bootcamp.  
The platform provides browsing of games, publishers, user profiles, and a contact system — all inside a responsive UI.

---

## 📌 Features

### 🏠 Home Page
- Hero section  
- Games listing (dynamic)  
- Popular games  
- Rating system  
- Categories  
- 3D controller placeholder  

### 👤 User Profile
- Avatar  
- About/Bio  
- Email  
- Twitch Link  
- Liked games list  

### 📝 About Page
- Vision  
- Mission  
- Community & Technical Focus  

### ✉️ Contact Page
- Form (subject + message)  
- Sends to backend  
- Support information  

### 📬 Admin Messages Dashboard
- User  
- Email  
- Subject  
- Message  
- Date/Time table  

### 🏢 Publishers System
- All publishers  
- Publisher detail  
- Add publisher form  

### 🔐 Authentication
- Login  
- Registration  

### ❌ Custom 404 Page
- Modern error design  
- Back to homepage button  

---

## 🛠️ Tech Stack
- **Backend:** Python 3.13, Django 5  
- **Frontend:** HTML, CSS, Bootstrap 5, Font Awesome  
- **Database:** SQLite3
